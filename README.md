# Oreoreo

🍪 一个沙雕的奥利奥定制生成器

Go and have a try 👉

## Preview

![](https://github.com/lioder/Oreoreo/raw/master/image/preview1)

![](https://github.com/lioder/Oreoreo/raw/master/image/preview2)

## Usage

- 加奥：加一层巧克力饼干层
- 加利：加一层奶油层
- 刮去一层：刮掉最下面的一层
- 制作：根据『配方』生成你点的奥利奥
